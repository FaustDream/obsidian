# 钉钉开发者平台接口

### 获取企业内部应用的accessToken

注意：企业内部应用调用前需要再钉钉管理后台创建一个应用来开放应用能调用那些接口的权限（或者找已经有该权限的应用）

[https://api.dingtalk.com/v1.0/oauth2/accessToken](https://api.dingtalk.com/v1.0/oauth2/accessToken)  post

```json

{
  "appKey" : "",
  "appSecret" : ""
}
```

在应用的基础信息中
Client ID (原 AppKey 和 SuiteKey) Client Secret (原 AppSecret 和 SuiteSecret)

```json

{
  "accessToken" : "fw8ef8we8f76e6f7s8dxxxx",
  "expireIn" : 7200
}
```

### 查询用户详情

[https://oapi.dingtalk.com/topapi/v2/user/get](https://oapi.dingtalk.com/topapi/v2/user/get) post

Query 参数：access_token

```json

{
        "language":"zh_CN",
        "userid":"zhangsan" //钉钉id
}
```
