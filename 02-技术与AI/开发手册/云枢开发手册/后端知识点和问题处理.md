- 历史日志，怎么看

```latex

压缩：

gzip file1.txt file2.txt file3.txt

每个文件都会分别生成 .gz 文件。

gzip 压缩后会替换原文件，若想保留原文件，可以使用 -k 选项（某些系统需要 GNU gzip 支持）：

gzip -k example.txt

解压：

gunzip 是 gzip 的别名，功能完全相同：

gzip -d example.txt.gz
gunzip example.txt.gz

执行后，会将 example.txt.gz 解压为 example.txt，并删除原来的 .gz 文件。

如果你希望解压后保留原 .gz 文件，可以使用 -k 选项（某些系统需要 GNU gzip 支持）：

gzip -dk example.txt.gz
```

- 业务规则发布不生效

```java

public Object handlePause(final Request request) {
    final SchedulerFacade schedulerFacade = SpringContextUtils.getBean(SchedulerFacade.class);
    final List<JobModel> groupJobs = schedulerFacade.findGroupJobs(SystemConstants.GROUP);
    if (!CollectionUtils.isEmpty(groupJobs)) {
        for (final JobModel groupJob : groupJobs) {
            final Boolean pauseJob = schedulerFacade.pauseJob(SystemConstants.GROUP, groupJob.getName());
            log.info("task pause status: {}", pauseJob);
        }
    }
    return request.getData().getBizObject();
}
```

```java

public Object handleResume(final Request request) {
    final SchedulerFacade schedulerFacade = ApplicationContextUtils.getBean(SchedulerFacade.class);
    final List<JobModel> groupJobs = schedulerFacade.findGroupJobs(SystemConstants.GROUP);
    if (!CollectionUtils.isEmpty(groupJobs)) {
        for (final JobModel groupJob : groupJobs) {
            final Boolean resumed = schedulerFacade.resumeJob(SystemConstants.GROUP, groupJob.getName());
            log.info("task resume status: {}", resumed);
        }
    }
    return request.getData().getBizObject();
}
```
