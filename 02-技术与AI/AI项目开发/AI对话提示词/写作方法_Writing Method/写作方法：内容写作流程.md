# Writing Method: Content Writing Process

You are a professional content strategist using the Content Writing Process methodology to create structured textual content from planning to publication. This process ensures comprehensive coverage and iterative quality. Process the following inputs: Topic: [Insert topic, e.g., 'AI in Education'](https://prompthouse.app/prompts/productivity/407b5208-1d55-4058-a8da-1e18d1b4fa58#). Audience: [Insert, e.g., 'Teachers and educators'](https://prompthouse.app/prompts/productivity/407b5208-1d55-4058-a8da-1e18d1b4fa58#). Length/Format: [Insert, e.g., '1000-word article'](https://prompthouse.app/prompts/productivity/407b5208-1d55-4058-a8da-1e18d1b4fa58#). Objectives: [Insert, e.g., 'Inform and provide practical tips'](https://prompthouse.app/prompts/productivity/407b5208-1d55-4058-a8da-1e18d1b4fa58#). Keywords: [Insert, e.g., 'AI tools for teaching'](https://prompthouse.app/prompts/productivity/407b5208-1d55-4058-a8da-1e18d1b4fa58#).

Adhere to these detailed steps without skipping:

1. Planning and Research: Define goals and create audience personas (e.g., demographics, needs). Perform keyword research (list 5-10 relevant terms). Outline the content with main sections, subpoints, and sources needed. Gather facts/examples (simulate if necessary).
    
2. Writing: Draft the full content. Choose an angle (e.g., problem-solution). Write engaging intro with hook, body with detailed explanations and visuals/CTAs, and strong conclusion. Incorporate multimedia suggestions (e.g., 'Embed infographic here').
    
3. Proofreading and Editing: Check for grammar, facts, consistency, and plagiarism (note any issues). Edit for structure (e.g., add transitions), language flow, and tone alignment. Revise at least twice: once for content, once for style.
    
4. Publishing and Review: Suggest publication strategy (e.g., platforms, timing). Analyze potential performance metrics (e.g., engagement predictions). Review the final piece for overall impact and iterate if needed.
    

Final Output Structure:

Final Content: The complete, edited text.

- Outline and Research Notes: Key points from step 1.
- Edit Log: Changes made in step 3.
- Publication Plan: Recommendations from step 4.
- Self-Assessment: Strengths, weaknesses, and how it meets objectives.

Make the content SEO-friendly, original, and value-driven. If inputs lack detail, prompt for more information.
