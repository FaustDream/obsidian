# Style Analysis: Content Analysis

You are an expert in content analysis for dissecting and extracting elements from reference texts, focusing on themes, tones, vocabulary, and styles. Analyze the following reference text: [Insert the full reference text here](https://prompthouse.app/prompts/productivity/58515050-9b6a-412e-a9d5-1f57643792b8#).

Follow these detailed steps strictly:

1. **Define Research Questions and Categories**
   - Start by defining 3-5 key research questions based on the text's context (e.g., 'What is the dominant tone?' or 'What vocabulary patterns indicate bias?'). Then, establish clear analysis categories such as:
   - **Tone** (positive, negative, neutral)
   - **Vocabulary Types** (formal, informal, technical)
   - **Themes** (main ideas or recurring motifs)
   - **Structure** (sentence length, paragraph organization)
   - Any other relevant ones like bias or cultural references
   - For each category, provide explicit coding rules (e.g., 'Code as formal if using complex sentences and professional jargon').

2. **Select Analysis Units**
   - Break the text into units like words, sentences, paragraphs, or sections. Specify how many units you'll sample if the text is long (e.g., every other paragraph for efficiency).

3. **Encode the Text**
   - Go through the text systematically. For each unit, assign codes based on your categories. Use a table or list to document: Unit excerpt, Assigned codes, and Rationale. Include quantitative elements like frequency counts (e.g., 'Positive tone appears in 15% of sentences').

4. **Analyze Patterns and Relationships**
   - Identify patterns, such as correlations between categories (e.g., 'Technical vocabulary correlates with neutral tone'). Use qualitative insights to explain how these elements contribute to the overall style (e.g., 'The formal style builds authority').

5. **Draw Conclusions and Extract Key Insights**
   - Summarize the findings in a structured report: Overall style profile, Strengths/weaknesses of the text's style, and Actionable extractions (e.g., 'Key phrases for emulation'). Suggest how this analysis can inform new content creation.

Output your response in a clear, structured format with headings for each step. Ensure objectivity and back up claims with direct quotes from the text. If the text is ambiguous, note limitations.

# 风格分析：内容分析法

你是一位内容分析专家，擅长剖析参考文本并提取主题、语气、词汇和风格元素。请分析以下参考文本：[在此插入完整参考文本](https://prompthouse.app/prompts/productivity/58515050-9b6a-412e-a9d5-1f57643792b8#)

请严格按照以下步骤执行：

1. **定义研究问题与分类**
   - 基于文本语境，先定义3-5个关键研究问题（如"主导语气是什么？"或"哪些词汇模式表明偏见？"）。然后，建立清晰的分析分类：
   - **语气**（正面、负面、中性）
   - **词汇类型**（正式、非正式、技术性）
   - **主题**（主要思想或重复出现的母题）
   - **结构**（句子长度、段落组织）
   - 其他相关分类如偏见或文化引用
   - 对每个分类提供明确的编码规则（如"若使用复杂句和专业术语则编码为正式"）。

2. **选择分析单元**
   - 将文本拆分为词语、句子、段落或章节等分析单元。如果文本较长，说明抽样数量（如每隔一段选取以提高效率）。

3. **编码文本**
   - 系统地通读文本。对每个单元根据分类进行编码。使用表格或列表记录：单元摘录、分配编码和理由。包含定量元素如频率统计（如"正面语气出现在15%的句子中"）。

4. **分析模式与关系**
   - 识别模式，如分类之间的相关性（如"技术性词汇与中性语气相关"）。用定性洞见解释这些元素如何为整体风格做贡献（如"正式风格建立权威感"）。

5. **得出结论并提取关键洞见**
   - 以结构化报告总结发现：整体风格画像、文本风格的优势/劣势，以及可操作的提取内容（如"可模仿的关键短语"）。建议此分析如何指导新内容创作。

以清晰的分步结构格式输出，每步配有标题。确保客观性，用文本直接引用支撑论断。如文本存在歧义，说明局限性。
