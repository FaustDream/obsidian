
### 1. Dribbble
- 网站：https://dribbble.com/
- 特点：
  - 有大量桌面端软件界面设计稿
  - 可以搜索关键词：`desktop app`、`dashboard`、`mac app ui`、`windows app`
  - 偏视觉展示，适合看风格趋势

### 2. Behance
- 网站：https://www.behance.net/
- 特点：
  - 很多完整项目展示，不只是单张界面
  - 能看到软件从登录页、主界面到交互流程的整体设计
  - 搜索：`desktop application ui`、`software ui design`

### 3. Mobbin
- 网站：https://mobbin.com/
- 特点：
  - 虽然以移动端为主，但也有部分 Web / 桌面产品参考
  - 适合看成熟产品的界面结构和交互模式

### 4. UI8
- 网站：https://ui8.net/
- 特点：
  - 提供高质量 UI 资源和桌面软件模板
  - 能看到很多专业级 SaaS、桌面端管理系统风格
  - 有些资源付费，但参考价值高

### 5. Awwwards
- 网站：https://www.awwwards.com/
- 特点：
  - 主要是网站设计灵感，但也能找到很多有“桌面软件风格”的 Web App 界面
  - 适合看高级、现代、动效感强的界面风格

### 6. Pttrns
- 网站：https://pttrns.com/
- 特点：
  - 主要偏移动端，但也可以参考模块化布局和视觉风格
  - 如果你想借鉴桌面端组件风格，也有参考意义

### 7. Figma Community
- 网站：https://www.figma.com/community
- 特点：
  - 有很多桌面应用 UI Kit、Windows/macOS 风格组件库
  - 可以直接查看甚至复用设计文件
  - 搜索：`desktop ui kit`、`macos app ui`、`windows ui`

### 8. Pinterest
- 网站：https://www.pinterest.com/
- 特点：
  - 搜索灵感非常方便
  - 可以快速看到大量桌面软件截图、概念设计、不同风格拼图
  - 搜索词建议：`desktop app design`、`admin desktop ui`

### 9. Lapa Ninja
- 网站：https://www.lapa.ninja/
- 特点：
  - 偏落地页和 SaaS 产品展示
  - 很多页面会嵌入产品后台、桌面端风格界面截图
  - 适合找现代简洁风格

### 10. SaaSFrame
- 网站：https://www.saasframe.io/
- 特点：
  - 收录很多 SaaS 产品界面
  - 虽然不完全是传统桌面软件，但非常适合参考“PC 软件风格”的后台和工具类产品设计
### 11. Microsoft Store / Apple 官网
- Microsoft Store：https://apps.microsoft.com/
- Apple（Mac App Store 相关）：
  - https://www.apple.com/mac-app-store/
- 特点：
  - 能直接看到真实软件截图
  - 更适合看实际落地的桌面应用，而不是概念稿

### 12. Product Hunt
- 网站：https://www.producthunt.com/
- 特点：
  - 很多新发布的软件产品会放界面截图或演示视频
  - 可以看到现在流行的生产力工具、AI 工具、桌面客户端设计风格

---

## 推荐搜索关键词
你可以直接搜这些词，更容易找到 PC 桌面软件界面：
- `desktop app ui`
- `desktop application design`
- `windows app ui`
- `mac app ui`
- `software dashboard ui`
- `enterprise software ui`
- `saas desktop interface`
