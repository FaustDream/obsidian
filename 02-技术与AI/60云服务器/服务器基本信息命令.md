### 关键软件检查
- **Node.js / npm / Rust / Cargo / Git**
  ```bash
  node -v 2>/dev/null || echo "NOT_INSTALLED"
  npm -v 2>/dev/null || echo "NOT_INSTALLED"
  rustc -V 2>/dev/null || echo "NOT_INSTALLED"
  cargo -V 2>/dev/null || echo "NOT_INSTALLED"
  git --version 2>/dev/null || echo "NOT_INSTALLED"
  ```
- **Nginx 是否已安装**
  ```bash
  nginx -v 2>&1 || echo "NGINX_NOT_INSTALLED"
  ```
- **是否支持 systemd**（用于设置开机自启）
  ```bash
  systemctl --version 2>/dev/null | head -1 || echo "NO_SYSTEMD"
  ```

### 宝塔相关

- **查看宝塔版本 / 默认信息**
  ```bash
  bt default 2>/dev/null || echo "BT_CMD_NOT_FOUND"
  ```
- **宝塔常用命令**
  ```bash
  启动 bt stop
  关闭 bt start
  重启 bt restart
  状态 bt status
  信息 bt default
  全部信息 bt
  禁止宝塔开机启动 systemctl disable bt
  运行宝塔开机启动 systemctl enable bt
  立即启动 systemctl start bt
  ```
### 端口与进程

- **检查常用端口占用**（重点看 `1420`、`80`、`443`、`8080`）
  ```bash
  ss -tlnp | grep -E ':(1420|80|443|8080)\b' || echo "端口均空闲"
  ```
- **彩色进程查看**
  ```bash
  htop
  ```
- **内存占用最高的进程**
  ```bash
  ps aux --sort=-%mem | head
  ```
- **实际内存占用最高的进程**
  ```bash
  ps aux --sort=-rss | head
  ```

#### 服务管理

### MySQL（MariaDB）

```bash
bt start mysql      # 启动
bt stop mysql       # 停止
bt restart mysql    # 重启
```

### PHP（php-fpm）

> 注意替换版本号：例如 PHP 8.1 使用 `81`

```bash
bt start php-fpm-81     # 启动
bt stop php-fpm-81      # 停止
bt restart php-fpm-81   # 重启
```

### 一键操作

```bash
bt start mysql && bt start php-fpm-81
bt stop mysql && bt stop php-fpm-81
bt restart mysql && bt restart php-fpm-81
```
