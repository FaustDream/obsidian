# 说出你最常用的算法，以及如何实现

作为Java开发工程师，我最常用的几种算法及其实现如下：

## 1. 排序算法 - 快速排序

**使用场景**：数据排序、TopK问题 **时间复杂度**：平均O(nlogn)，最坏O(n²)

```java
public void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        int pivot = partition(arr, low, high);
        quickSort(arr, low, pivot - 1);
        quickSort(arr, pivot + 1, high);
    }
}

private int partition(int[] arr, int low, int high) {
    int pivot = arr[high];
    int i = low - 1;
    
    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            swap(arr, i, j);
        }
    }
    swap(arr, i + 1, high);
    return i + 1;
}
```

## 2. 二分查找

**使用场景**：有序数组查找、数据库索引原理 **时间复杂度**：O(logn)

```java
public int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

## 3. 深度优先搜索(DFS)

**使用场景**：树遍历、图遍历、回溯问题 **应用实例**：文件系统遍历、权限检查

```java
public void dfs(TreeNode root, List<Integer> result) {
    if (root == null) return;
    
    result.add(root.val);  // 前序遍历
    dfs(root.left, result);
    dfs(root.right, result);
}

// 图的DFS
public void dfsGraph(int[][] graph, int start, boolean[] visited) {
    visited[start] = true;
    System.out.println(start);
    
    for (int i = 0; i < graph[start].length; i++) {
        if (graph[start][i] == 1 && !visited[i]) {
            dfsGraph(graph, i, visited);
        }
    }
}
```

## 4. 广度优先搜索(BFS)

**使用场景**：最短路径、层次遍历 **应用实例**：社交网络关系链、推荐系统

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;
    
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    
    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> level = new ArrayList<>();
        
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

## 5. 动态规划

**使用场景**：优化问题、缓存计算结果 **应用实例**：用户行为预测、资源分配

```java
// 斐波那契数列 - 自底向上
public int fibonacci(int n) {
    if (n <= 1) return n;
    
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = 1;
    
    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    return dp[n];
}

// 最长公共子序列
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i-1) == text2.charAt(j-1)) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }
    return dp[m][n];
}
```

## 扩展知识点

**算法选择的考虑因素**：

- 时间复杂度和空间复杂度
- 数据规模和特征
- 稳定性要求
- 实现复杂度

**在实际项目中的应用**：

- 排序算法：用户列表排序、商品推荐排序
- 搜索算法：全文检索、地理位置搜索
- 动态规划：缓存策略、路径优化
- 图算法：社交网络分析、依赖关系解析

**性能优化技巧**：

- 选择合适的数据结构
- 避免重复计算
- 充分利用缓存
- 考虑并行处理

这些算法在日常开发中经常遇到，掌握它们的实现原理和应用场景对提升代码质量和解决复杂问题很有帮助。
