# 动态规划的原理是什么 解决哪些问题

动态规划（Dynamic Programming，简称DP）是一种算法设计技术，用于解决具有重叠子问题和最优子结构特性的问题。

## 基本原理

**核心思想**：将复杂问题分解为更小的子问题，通过保存子问题的解来避免重复计算，从而提高算法效率。

**两个关键特性**：

1. **重叠子问题**：在求解过程中，同样的子问题会被多次计算
2. **最优子结构**：问题的最优解包含子问题的最优解

**实现方式**：

- **自顶向下（记忆化搜索）**：递归 + 缓存
- **自底向上（填表法）**：迭代填充状态表

## 适用问题类型

### 1. 最优化问题

- **背包问题**：01背包、完全背包、多重背包
- **路径问题**：最短路径、最大路径和
- **股票买卖**：最大利润问题

### 2. 计数问题

- **斐波那契数列**：经典入门问题
- **爬楼梯**：有多少种方法到达第n层
- **不同路径**：网格中从起点到终点的路径数

### 3. 判定问题

- **子序列问题**：最长公共子序列（LCS）
- **回文问题**：最长回文子串
- **字符串匹配**：编辑距离

## 经典例题解析

### 斐波那契数列

```java
// 传统递归（效率低）
public int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}

// 动态规划优化
public int fibDP(int n) {
    if (n <= 1) return n;
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = 1;
    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    return dp[n];
}
```

### 01背包问题

```java
public int knapsack(int[] weights, int[] values, int capacity) {
    int n = weights.length;
    int[][] dp = new int[n + 1][capacity + 1];
    
    for (int i = 1; i <= n; i++) {
        for (int w = 1; w <= capacity; w++) {
            if (weights[i-1] <= w) {
                dp[i][w] = Math.max(
                    dp[i-1][w], // 不选当前物品
                    dp[i-1][w-weights[i-1]] + values[i-1] // 选当前物品
                );
            } else {
                dp[i][w] = dp[i-1][w];
            }
        }
    }
    return dp[n][capacity];
}
```

## 解题步骤

1. **确定状态定义**：dp[i]表示什么含义
2. **找出状态转移方程**：dp[i]与前面状态的关系
3. **确定初始条件**：边界值的处理
4. **确定计算顺序**：避免使用未计算的状态

## 优化技巧

### 空间优化

很多DP问题可以通过滚动数组或状态压缩来优化空间复杂度：

```java
// 空间优化的斐波那契
public int fibOptimized(int n) {
    if (n <= 1) return n;
    int prev2 = 0, prev1 = 1;
    for (int i = 2; i <= n; i++) {
        int curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

## 与其他算法的区别

- **与分治算法**：分治没有重叠子问题，DP有
- **与贪心算法**：贪心每步都做局部最优选择，DP考虑所有可能
- **与回溯算法**：回溯会撤销选择，DP保存中间结果

动态规划是解决优化问题的强大工具，掌握其思想和常见模式对编程面试和实际开发都非常重要。关键在于识别问题的DP特性，正确定义状态和转移方程。
