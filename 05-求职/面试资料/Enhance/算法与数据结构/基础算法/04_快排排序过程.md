# 快排排序过程

快速排序（QuickSort）是一种高效的分治排序算法，我来详细解释其排序过程：

## 核心思想

快速排序采用"分治"策略，通过选择一个基准元素（pivot），将数组分为两部分：小于基准的元素和大于基准的元素，然后递归地对这两部分进行排序。

## 详细排序过程

### 1. 选择基准元素（Pivot）

```java
// 常见的选择方式：选择第一个元素作为基准
int pivot = arr[low];
```

### 2. 分区（Partition）过程

这是快排的核心步骤，目标是将数组重新排列，使得：

- 所有小于基准的元素都在基准左边
- 所有大于基准的元素都在基准右边

```java
public int partition(int[] arr, int low, int high) {
    int pivot = arr[low];  // 选择第一个元素作为基准
    int i = low, j = high;
    
    while (i < j) {
        // 从右向左找第一个小于基准的元素
        while (i < j && arr[j] >= pivot) {
            j--;
        }
        if (i < j) {
            arr[i++] = arr[j];  // 将小元素移到左边
        }
        
        // 从左向右找第一个大于基准的元素
        while (i < j && arr[i] <= pivot) {
            i++;
        }
        if (i < j) {
            arr[j--] = arr[i];  // 将大元素移到右边
        }
    }
    
    arr[i] = pivot;  // 基准元素放到正确位置
    return i;        // 返回基准元素的最终位置
}
```

### 3. 递归排序

```java
public void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        int pivotIndex = partition(arr, low, high);
        quickSort(arr, low, pivotIndex - 1);   // 递归排序左半部分
        quickSort(arr, pivotIndex + 1, high);  // 递归排序右半部分
    }
}
```

## 图解示例

假设有数组 `[6, 1, 2, 7, 9, 3, 4, 5, 10, 8]`，以第一个元素6作为基准：

```
初始: [6, 1, 2, 7, 9, 3, 4, 5, 10, 8]
      ↑                           ↑
      i                           j

第一轮分区后: [5, 1, 2, 4, 3, 6, 7, 9, 10, 8]
                              ↑
                           基准位置

递归处理左半部分: [5, 1, 2, 4, 3]
递归处理右半部分: [7, 9, 10, 8]
```

## 完整实现

```java
public class QuickSort {
    public void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pivotIndex = partition(arr, low, high);
            quickSort(arr, low, pivotIndex - 1);
            quickSort(arr, pivotIndex + 1, high);
        }
    }
    
    private int partition(int[] arr, int low, int high) {
        int pivot = arr[low];
        int i = low, j = high;
        
        while (i < j) {
            while (i < j && arr[j] >= pivot) j--;
            if (i < j) arr[i++] = arr[j];
            
            while (i < j && arr[i] <= pivot) i++;
            if (i < j) arr[j--] = arr[i];
        }
        
        arr[i] = pivot;
        return i;
    }
}
```

## 扩展知识点

### 时间复杂度

- **最好情况**: O(n log n) - 每次都能平均分割
- **平均情况**: O(n log n)
- **最坏情况**: O(n²) - 每次分割都极不平衡（如已排序数组）

### 空间复杂度

- **平均情况**: O(log n) - 递归调用栈
- **最坏情况**: O(n) - 递归深度达到n

### 优化策略

1. **三数取中法选择基准**：避免最坏情况
2. **小数组使用插入排序**：当子数组长度小于某个阈值时切换
3. **三路快排**：处理大量重复元素的情况
4. **迭代实现**：避免递归栈溢出

### 稳定性

快速排序是**不稳定**的排序算法，因为在分区过程中，相等元素的相对位置可能发生改变。

快速排序因其优秀的平均性能和原地排序的特性，在实际应用中被广泛使用，是Java标准库中Arrays.sort()对基本类型数组排序的核心算法之一。
