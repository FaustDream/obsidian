# 两个数组怎么最快去取交集

在Java中，有多种方式可以取两个数组的交集，我来介绍几种最常用且高效的方法：

## 1. 使用HashSet（推荐方式）

```java
import java.util.*;

public class ArrayIntersection {
    public static int[] intersection(int[] nums1, int[] nums2) {
        Set<Integer> set1 = new HashSet<>();
        Set<Integer> resultSet = new HashSet<>();
        
        // 将第一个数组元素加入set1
        for (int num : nums1) {
            set1.add(num);
        }
        
        // 遍历第二个数组，如果在set1中存在则加入结果集
        for (int num : nums2) {
            if (set1.contains(num)) {
                resultSet.add(num);
            }
        }
        
        // 转换为数组返回
        return resultSet.stream().mapToInt(Integer::intValue).toArray();
    }
}
```

**时间复杂度：O(m + n)**  
**空间复杂度：O(min(m, n))**

## 2. 使用双指针（适用于已排序数组）

```java
public static int[] intersectionSorted(int[] nums1, int[] nums2) {
    Arrays.sort(nums1);
    Arrays.sort(nums2);
    
    List<Integer> result = new ArrayList<>();
    int i = 0, j = 0;
    
    while (i < nums1.length && j < nums2.length) {
        if (nums1[i] < nums2[j]) {
            i++;
        } else if (nums1[i] > nums2[j]) {
            j++;
        } else {
            // 避免重复元素
            if (result.isEmpty() || result.get(result.size() - 1) != nums1[i]) {
                result.add(nums1[i]);
            }
            i++;
            j++;
        }
    }
    
    return result.stream().mapToInt(Integer::intValue).toArray();
}
```

**时间复杂度：O(m log m + n log n)**（排序开销）  
**空间复杂度：O(1)**（不考虑结果数组）

## 3. 使用Java 8 Stream API（简洁写法）

```java
public static int[] intersectionStream(int[] nums1, int[] nums2) {
    Set<Integer> set2 = Arrays.stream(nums2)
                             .boxed()
                             .collect(Collectors.toSet());
    
    return Arrays.stream(nums1)
                 .distinct()
                 .filter(set2::contains)
                 .toArray();
}
```

## 性能对比和选择建议

**最快方案：HashSet方法**

- 适用于大部分场景
- 时间复杂度最优
- 代码简洁易懂

**双指针方法适用场景：**

- 数组已经排序
- 内存空间受限
- 需要保持元素顺序

## 扩展知识点

### 1. 如果允许重复元素的交集？

```java
public static int[] intersectionWithDuplicates(int[] nums1, int[] nums2) {
    Map<Integer, Integer> map = new HashMap<>();
    List<Integer> result = new ArrayList<>();
    
    // 统计nums1中每个元素出现次数
    for (int num : nums1) {
        map.put(num, map.getOrDefault(num, 0) + 1);
    }
    
    // 遍历nums2，如果在map中存在且计数>0，则加入结果
    for (int num : nums2) {
        if (map.getOrDefault(num, 0) > 0) {
            result.add(num);
            map.put(num, map.get(num) - 1);
        }
    }
    
    return result.stream().mapToInt(Integer::intValue).toArray();
}
```

### 2. 处理大数据量的优化

- 选择较小的数组建立HashSet
- 使用BitSet处理数值范围较小的整数
- 考虑使用布隆过滤器进行预过滤

### 3. 多个数组求交集

```java
public static int[] multipleArraysIntersection(int[]... arrays) {
    if (arrays.length == 0) return new int[0];
    
    Set<Integer> result = new HashSet<>();
    for (int num : arrays[0]) {
        result.add(num);
    }
    
    for (int i = 1; i < arrays.length; i++) {
        Set<Integer> currentSet = new HashSet<>();
        for (int num : arrays[i]) {
            currentSet.add(num);
        }
        result.retainAll(currentSet);
    }
    
    return result.stream().mapToInt(Integer::intValue).toArray();
}
```

这些方法各有优势，在实际开发中建议根据数据特点选择最合适的方案。
