# 说说如何维护堆？大根堆和小跟堆的插入删除维护？

我来详细解释一下堆的维护，包括大根堆和小根堆的插入删除操作。

## 堆的基本概念

堆是一种完全二叉树，分为两种类型：

- **大根堆（最大堆）**：父节点的值总是大于或等于子节点的值
- **小根堆（最小堆）**：父节点的值总是小于或等于子节点的值

## 堆的维护操作

### 1. 向上调整（Heapify Up）

当在堆底插入新元素时，需要向上调整以维持堆的性质。

```java
// 大根堆向上调整
private void heapifyUp(int[] heap, int index) {
    while (index > 0) {
        int parentIndex = (index - 1) / 2;
        if (heap[index] <= heap[parentIndex]) {
            break; // 已满足大根堆性质
        }
        swap(heap, index, parentIndex);
        index = parentIndex;
    }
}

// 小根堆向上调整
private void heapifyUpMin(int[] heap, int index) {
    while (index > 0) {
        int parentIndex = (index - 1) / 2;
        if (heap[index] >= heap[parentIndex]) {
            break; // 已满足小根堆性质
        }
        swap(heap, index, parentIndex);
        index = parentIndex;
    }
}
```

### 2. 向下调整（Heapify Down）

当删除堆顶元素时，需要向下调整以维持堆的性质。

```java
// 大根堆向下调整
private void heapifyDown(int[] heap, int index, int size) {
    while (true) {
        int largest = index;
        int leftChild = 2 * index + 1;
        int rightChild = 2 * index + 2;
        
        // 找到父节点和子节点中的最大值
        if (leftChild < size && heap[leftChild] > heap[largest]) {
            largest = leftChild;
        }
        if (rightChild < size && heap[rightChild] > heap[largest]) {
            largest = rightChild;
        }
        
        if (largest == index) {
            break; // 已满足大根堆性质
        }
        
        swap(heap, index, largest);
        index = largest;
    }
}

// 小根堆向下调整
private void heapifyDownMin(int[] heap, int index, int size) {
    while (true) {
        int smallest = index;
        int leftChild = 2 * index + 1;
        int rightChild = 2 * index + 2;
        
        // 找到父节点和子节点中的最小值
        if (leftChild < size && heap[leftChild] < heap[smallest]) {
            smallest = leftChild;
        }
        if (rightChild < size && heap[rightChild] < heap[smallest]) {
            smallest = rightChild;
        }
        
        if (smallest == index) {
            break; // 已满足小根堆性质
        }
        
        swap(heap, index, smallest);
        index = smallest;
    }
}
```

## 插入操作

### 大根堆插入

```java
public void insertMaxHeap(int[] heap, int[] size, int value) {
    // 在堆尾插入新元素
    heap[size[0]] = value;
    size[0]++;
    
    // 向上调整
    heapifyUp(heap, size[0] - 1);
}
```

### 小根堆插入

```java
public void insertMinHeap(int[] heap, int[] size, int value) {
    // 在堆尾插入新元素
    heap[size[0]] = value;
    size[0]++;
    
    // 向上调整
    heapifyUpMin(heap, size[0] - 1);
}
```

## 删除操作

### 大根堆删除（删除堆顶最大元素）

```java
public int deleteMaxHeap(int[] heap, int[] size) {
    if (size[0] == 0) {
        throw new RuntimeException("堆为空");
    }
    
    int max = heap[0]; // 保存要删除的最大值
    
    // 将最后一个元素移到堆顶
    heap[0] = heap[size[0] - 1];
    size[0]--;
    
    // 向下调整
    if (size[0] > 0) {
        heapifyDown(heap, 0, size[0]);
    }
    
    return max;
}
```

### 小根堆删除（删除堆顶最小元素）

```java
public int deleteMinHeap(int[] heap, int[] size) {
    if (size[0] == 0) {
        throw new RuntimeException("堆为空");
    }
    
    int min = heap[0]; // 保存要删除的最小值
    
    // 将最后一个元素移到堆顶
    heap[0] = heap[size[0] - 1];
    size[0]--;
    
    // 向下调整
    if (size[0] > 0) {
        heapifyDownMin(heap, 0, size[0]);
    }
    
    return min;
}
```

## 完整的堆实现示例

```java
public class Heap {
    private int[] array;
    private int size;
    private int capacity;
    private boolean isMaxHeap;
    
    public Heap(int capacity, boolean isMaxHeap) {
        this.capacity = capacity;
        this.array = new int[capacity];
        this.size = 0;
        this.isMaxHeap = isMaxHeap;
    }
    
    public void insert(int value) {
        if (size >= capacity) {
            throw new RuntimeException("堆已满");
        }
        
        array[size] = value;
        size++;
        
        if (isMaxHeap) {
            heapifyUp(size - 1);
        } else {
            heapifyUpMin(size - 1);
        }
    }
    
    public int delete() {
        if (size == 0) {
            throw new RuntimeException("堆为空");
        }
        
        int result = array[0];
        array[0] = array[size - 1];
        size--;
        
        if (size > 0) {
            if (isMaxHeap) {
                heapifyDown(0);
            } else {
                heapifyDownMin(0);
            }
        }
        
        return result;
    }
    
    // 辅助方法
    private void swap(int i, int j) {
        int temp = array[i];
        array[i] = array[j];
        array[j] = temp;
    }
}
```

## 时间复杂度分析

- **插入操作**：O(log n) - 最多需要向上调整到根节点
- **删除操作**：O(log n) - 最多需要向下调整到叶子节点
- **查找最值**：O(1) - 堆顶就是最值

## 相关扩展知识

1. **堆排序**：利用大根堆可以实现升序排序，时间复杂度O(n log n)
2. **优先队列**：Java中的PriorityQueue就是基于堆实现的
3. **Top K问题**：使用小根堆可以高效解决Top K最大元素问题
4. **合并K个有序链表**：可以使用小根堆来维护每个链表的当前最小节点

堆的关键在于理解其完全二叉树的性质，以及如何通过向上和向下调整来维护堆的有序性。
