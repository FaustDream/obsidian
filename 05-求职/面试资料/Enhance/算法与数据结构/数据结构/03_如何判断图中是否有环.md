# 如何判断图中是否有环？

判断图中是否有环是一个经典的图论问题，根据图的类型（有向图或无向图），我们有不同的解决方案。

## 无向图环检测

### 1. DFS深度优先搜索法

```java
public class UndirectedGraphCycle {
    private boolean[] visited;
    private List<List<Integer>> adj;
    
    public boolean hasCycle(int V, List<List<Integer>> adj) {
        this.adj = adj;
        visited = new boolean[V];
        
        // 检查每个连通分量
        for (int i = 0; i < V; i++) {
            if (!visited[i]) {
                if (dfs(i, -1)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    private boolean dfs(int node, int parent) {
        visited[node] = true;
        
        for (int neighbor : adj.get(node)) {
            if (!visited[neighbor]) {
                if (dfs(neighbor, node)) {
                    return true;
                }
            } else if (neighbor != parent) {
                // 访问到已访问的节点且不是父节点，说明有环
                return true;
            }
        }
        return false;
    }
}
```

### 2. 并查集（Union-Find）法

```java
public class UnionFind {
    private int[] parent;
    private int[] rank;
    
    public UnionFind(int n) {
        parent = new int[n];
        rank = new int[n];
        for (int i = 0; i < n; i++) {
            parent[i] = i;
        }
    }
    
    public int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]); // 路径压缩
        }
        return parent[x];
    }
    
    public boolean union(int x, int y) {
        int rootX = find(x);
        int rootY = find(y);
        
        if (rootX == rootY) {
            return false; // 已经在同一集合中，形成环
        }
        
        // 按秩合并
        if (rank[rootX] < rank[rootY]) {
            parent[rootX] = rootY;
        } else if (rank[rootX] > rank[rootY]) {
            parent[rootY] = rootX;
        } else {
            parent[rootY] = rootX;
            rank[rootX]++;
        }
        return true;
    }
    
    public boolean hasCycle(int V, int[][] edges) {
        UnionFind uf = new UnionFind(V);
        for (int[] edge : edges) {
            if (!uf.union(edge[0], edge[1])) {
                return true;
            }
        }
        return false;
    }
}
```

## 有向图环检测

### 1. DFS + 递归栈法

```java
public class DirectedGraphCycle {
    private boolean[] visited;
    private boolean[] recStack; // 递归栈，记录当前路径
    private List<List<Integer>> adj;
    
    public boolean hasCycle(int V, List<List<Integer>> adj) {
        this.adj = adj;
        visited = new boolean[V];
        recStack = new boolean[V];
        
        for (int i = 0; i < V; i++) {
            if (!visited[i]) {
                if (dfs(i)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    private boolean dfs(int node) {
        visited[node] = true;
        recStack[node] = true;
        
        for (int neighbor : adj.get(node)) {
            if (!visited[neighbor]) {
                if (dfs(neighbor)) {
                    return true;
                }
            } else if (recStack[neighbor]) {
                // 在当前路径中遇到已访问节点，说明有环
                return true;
            }
        }
        
        recStack[node] = false; // 回溯时移除
        return false;
    }
}
```

### 2. 拓扑排序法（Kahn算法）

```java
public class TopologicalSort {
    public boolean hasCycle(int V, List<List<Integer>> adj) {
        int[] indegree = new int[V];
        
        // 计算每个节点的入度
        for (int i = 0; i < V; i++) {
            for (int neighbor : adj.get(i)) {
                indegree[neighbor]++;
            }
        }
        
        Queue<Integer> queue = new LinkedList<>();
        // 将入度为0的节点加入队列
        for (int i = 0; i < V; i++) {
            if (indegree[i] == 0) {
                queue.offer(i);
            }
        }
        
        int processedNodes = 0;
        while (!queue.isEmpty()) {
            int node = queue.poll();
            processedNodes++;
            
            for (int neighbor : adj.get(node)) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) {
                    queue.offer(neighbor);
                }
            }
        }
        
        // 如果处理的节点数小于总节点数，说明有环
        return processedNodes != V;
    }
}
```

## 复杂度分析

|方法|时间复杂度|空间复杂度|适用场景|
|---|---|---|---|
|DFS|O(V + E)|O(V)|通用，直观易懂|
|并查集|O(E·α(V))|O(V)|无向图，动态添加边|
|拓扑排序|O(V + E)|O(V)|有向图，同时可得拓扑序|

## 扩展知识点

1. **强连通分量**：有向图中的环检测可以扩展到寻找强连通分量（SCC）
2. **最小生成树**：Kruskal算法使用并查集避免环的形成
3. **图着色**：二分图判断也使用类似的DFS思想
4. **依赖关系**：在项目管理中，拓扑排序用于检测循环依赖

这些算法在实际开发中广泛应用于依赖管理、死锁检测、编译器优化等场景。
