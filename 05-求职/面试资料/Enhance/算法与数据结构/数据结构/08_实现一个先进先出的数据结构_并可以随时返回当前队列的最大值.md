# 实现一个先进先出的数据结构，并可以随时返回当前队列的最大值



## 核心思路

使用**双端队列(Deque)辅助法**：

- 主队列：存储所有元素，维护FIFO顺序
- 辅助双端队列：存储可能的最大值候选，维护单调递减序列## 算法详解

import java.util.*;

/**
 * 支持先进先出并能随时返回最大值的队列实现
 * 所有操作的时间复杂度都是O(1)
 */
public class MaxQueue {
    // 主队列：存储所有元素，维护FIFO顺序
    private Queue<Integer> queue;
    
    // 辅助双端队列：存储可能的最大值候选
    // 保持单调递减序列，队首始终是当前最大值
    private Deque<Integer> maxDeque;
    
    public MaxQueue() {
        queue = new LinkedList<>();
        maxDeque = new ArrayDeque<>();
    }
    
    /**
     * 入队操作
     * @param value 要入队的值
     */
    public void enqueue(int value) {
        // 1. 正常入队
        queue.offer(value);
        
        // 2. 维护辅助队列的单调递减性质
        // 从队尾移除所有小于当前值的元素
        while (!maxDeque.isEmpty() && maxDeque.peekLast() < value) {
            maxDeque.pollLast();
        }
        
        // 3. 将当前值加入辅助队列尾部
        maxDeque.offerLast(value);
    }
    
    /**
     * 出队操作
     * @return 出队的元素，如果队列为空返回-1
     */
    public int dequeue() {
        if (queue.isEmpty()) {
            return -1;
        }
        
        // 1. 正常出队
        int value = queue.poll();
        
        // 2. 如果出队的元素是当前最大值，也要从辅助队列中移除
        if (!maxDeque.isEmpty() && maxDeque.peekFirst() == value) {
            maxDeque.pollFirst();
        }
        
        return value;
    }
    
    /**
     * 获取当前队列的最大值
     * @return 最大值，如果队列为空返回-1
     */
    public int getMax() {
        if (maxDeque.isEmpty()) {
            return -1;
        }
        return maxDeque.peekFirst();
    }
    
    /**
     * 判断队列是否为空
     */
    public boolean isEmpty() {
        return queue.isEmpty();
    }
    
    /**
     * 获取队列大小
     */
    public int size() {
        return queue.size();
    }
    
    /**
     * 获取队列内容（用于调试）
     */
    public String toString() {
        return "Queue: " + queue.toString() + 
               ", MaxDeque: " + maxDeque.toString() + 
               ", CurrentMax: " + getMax();
    }
}

/**
 * 算法分析：
 * 
 * 时间复杂度：
 * - enqueue(): O(1) 均摊时间复杂度
 * - dequeue(): O(1) 时间复杂度  
 * - getMax(): O(1) 时间复杂度
 * 
 * 空间复杂度：O(n)，其中n是队列中元素的数量
 * 
 * 核心思想：
 * 1. 主队列维护FIFO顺序
 * 2. 辅助双端队列维护单调递减序列，确保队首始终是最大值
 * 3. 入队时从队尾移除所有小于当前值的元素
 * 4. 出队时如果移除的是最大值，同时从辅助队列移除
 */



### 1. **数据结构设计**

- **主队列**：使用 `LinkedList` 实现，维护标准的FIFO顺序
- **辅助双端队列**：使用 `ArrayDeque` 实现，维护单调递减序列

### 2. **核心操作原理**

**入队(enqueue)操作：**

- 元素正常加入主队列
- 在辅助队列中，从队尾移除所有小于当前值的元素
- 将当前值加入辅助队列尾部

**出队(dequeue)操作：**

- 从主队列正常出队
- 如果出队元素恰好是当前最大值，也要从辅助队列队首移除

**获取最大值(getMax)：**

- 直接返回辅助队列的队首元素

### 3. **时间复杂度分析**

虽然入队操作中有while循环，但每个元素最多被移除一次，所以：

- **入队**：O(1) 均摊时间复杂度
- **出队**：O(1) 时间复杂度
- **获取最大值**：O(1) 时间复杂度
