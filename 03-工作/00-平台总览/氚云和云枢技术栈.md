# 氚云和云枢技术栈

##### 氚云

后端：.NET Framework，版本：.NET Framework 4.6.2 
前端：JavaScript，版本：ECMAScript 5 / ECMAScript 2009 
数据库：MySQL 

##### 云枢

前端

- 框架：Vue 2、Vuex、Vue Router、Vue-I18n

- 语言：TypeScript 4、Less 3

- 主要组件：Lerna、RxJS、Axios、crypto-js
 后端

- 语言：Java （JDK 8）

- 框架和库

- Spring Boot: 2.5.x

- Spring Security

- Spring OAuth2

- Tomcat (嵌入式版, 由Spring Boot 引入)

- Spring Data JPA(Hibernate); Spring Data JDBC; HikariCP

- Spring Data Redis; Redisson

- Quartz

- SLF4J+ Logback

- Swagger
 中间件

- 数据库 (MySQL/Oracle/SqlServer/达梦/自定义扩展实现其它数据库支持)

- Redis

- Nginx
