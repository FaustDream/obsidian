# 云枢系统接口

1、`/api/runtime/query/list` 获取整个表单数据

- 如果不清楚可以找个云枢表格打开F12，查找网络找到list方法

- queryCondition 是写过滤条件

- queryCode和schemaCode 必填是对应表单模型层编码

```javascript

    let treeList = await axios.post("/api/runtime/query/list", {
      mobile: false,
      queryCode: "taskElement",
      schemaCode: "taskElement",
      queryVersion: 1,
      queryCondition: [
        [
          [
            {
              "queryFilterType": "Eq",
              "propertyCode": "businessDomain",
              "propertyType": 14,
              "propertyValue": "ZHJD"
            } 
          ]
        ]
      ],
      showTotal: true,
      view: {
        type: "TREE",
        depth: 1,
        parentRef: "#",
      }
    });
```
