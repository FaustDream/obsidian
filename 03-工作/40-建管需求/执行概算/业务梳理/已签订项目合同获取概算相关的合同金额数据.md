### 🧮 合同金额类字段

| 

字段名称
 

来源表别名
 

原始字段名/子查询
 

计算逻辑
 

说明
|
| 

`acontractProjectAmount`
 

a, d
 

`a.projTZJR`, `d.projTZJR1`, `a.planBasicPrepaid`, `a.planSpreadsPrepaid`
 

`SUM(a.projTZJR) + SUM(d.projTZJR1) - SUM(a.planBasicPrepaid) - SUM(a.planSpreadsPrepaid)`
 

合同金额减去计划中预备费用
|
| 

`contractProjTotalSettleAmt`
 

c, a
 

`c.settlementAmount`, `a.actualBasicPrepaid`, `a.ActualSpreadsPrepaid`
 

`c.settlementAmount - SUM(a.actualBasicPrepaid) - SUM(a.ActualSpreadsPrepaid)`
 

台账累计结算金额减去实际支付预备费用
|
| 

`contractProjEstSettleAmount`
 

h, c, a, d
 

`h.contractState`, `c.settlementAmount`, `a.projTZJR`, `d.projTZJR1`
 

`IF h.contractState = '已结算' THEN c.settlementAmount ELSE SUM(a.projTZJR) + SUM(d.projTZJR1)`
 

如果已结算，取台账金额，否则为原始金额之和
|
