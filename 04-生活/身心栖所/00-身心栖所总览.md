# 身心栖所总览

这个目录只做一件事：帮助我把身心状态、现实行动、职业方向和长期复盘放回一个清晰系统里。

## 先看什么

每天只需要优先看这几处：

1. [每日三锚点](10-行动系统/每日三锚点.md)：当天应该做什么，如何获得现实正反馈。
2. [30天行动记录](10-行动系统/30天行动记录/)：每天执行和复盘的记录区。
3. [对自己的看法](20-自我认知/对自己的看法.md)：当前最完整的自我画像和近期觉察。
4. [根因链与解锁序列](20-自我认知/HUMAN-整合分析/05-根因链与解锁序列.md)：当我又陷入混乱时，用来重新理解问题主线。
5. [职业规划](职业规划.md)：职业、考研、技术路线相关判断。

## 目录规则

- `10-行动系统`：每天要执行、打勾、复盘的内容。优先级最高。
- `20-自我认知`：关于我是谁、为什么卡住、核心问题是什么。
- `30-职业发展`：工作、考研、跳槽、技术成长相关材料。
- `40-能力训练`：表达、学习、工具依赖等单项能力改进。
- `80-工具与提示词`：AI 提示词、Life OS、HUMAN 模型等可复用工具。
- `90-归档`：旧版本、历史生成稿、暂时不作为主线阅读的材料。

## 主次判断

主文档是能直接指导我今天行动、理解当前状态、决定近期方向的文档。旧版本、长篇生成稿、提示词、重复评估报告都不作为每天阅读材料。

当前主线是：

```text
每日三锚点
-> 30天行动记录
-> 对自己的看法
-> HUMAN-整合分析/05-根因链与解锁序列
-> 职业规划
```

## 使用原则

不要每天重新翻全部文档。每天只完成一个现实动作，留下一个证据，再写一句复盘。这个目录不是用来制造更多焦虑的，而是用来减少混乱、降低启动难度、帮助我把注意力放回现实行动。
