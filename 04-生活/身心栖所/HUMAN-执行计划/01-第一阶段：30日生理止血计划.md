# 第一阶段：30日生理止血计划

> 周期：第1-30天
> 目标：不要求进步，只需要停止对身体的继续破坏
> 原则：这是你整个恢复过程中最重要的30天。不需要意志力，需要环境设计。

---

## 一、本阶段的总目标

| 目标维度 | 当前值 | 30天后可预期值 | 判断标准 |
|---------|-------|--------------|---------|
| 睡眠 | 深夜不规律(2-3点睡) | 22:30-23:00入睡 | 连续7天记录 |
| 饮食 | 外卖为主 | 外卖次数减半 | 外卖订单截图 |

| 网文/游戏 | 日均3-5小时 | ≤1小时/天 | 屏幕时间 |
| 工作摸鱼 | 持续 | 工作时长完成率提升 | 自我记录 |
| 甲亢治疗 | 服药但不规律 | 规律服药+复诊1次 | 药盒+挂号记录 |

## 二、为什么你当前的状态需要"止血"而非"进步"

你的身体处于**持续失能**状态。这不是比喻，是生理事实。

### 你的身体当前的能量预算

```
可支配总能量(每日) = 100单位

正常健康人分配：
    工作：40单位
    生活自理：20单位
    学习/社交/运动：30单位
    储备/缓冲：10单位

你当前分配(估计)：
    工作(最低程度维持)：45单位 ← 超支了
    生活自理：15单位
    焦虑内耗：25单位 ← 严重浪费
    对抗欲望(抗拒网文/色情)：15单位
    剩余可用于学习/运动的能量：0单位
```

**这就是你"想学但学不动"的物理真相。** 你的能量预算表里没有给"进步"留出空间。强行逼自己学习，等于用透支信用卡的方式支付电费——总有一天会爆。

**止血 = 减少浪费 = 重新分配能量。** 你不需要节能(因为已经是最低功耗了)，你需要"降低无效支出"以腾出空间给恢复。

## 三、"时间余地"设计原则

本阶段计划的核心设计理念是"至少50%的缓冲空间"。

| 每日时间块 | 核心活动 | 时长 | 缓冲说明 |
|-----------|---------|------|---------|
| 07:30-09:00 | 起床→通勤→上班 | 1.5h | 如果起晚了，跳过拉伸直接出门 |
| 09:00-12:00 | 工作(上午) | 3h | 完成最低任务即可，不要求高效 |
| 12:00-13:30 | 午餐+下午调度 | 1.5h | 允许饭后刷15分钟手机(可控豁免) |
| 13:30-18:00 | 工作(下午) | 4.5h | 如果太困，允许趴桌15分钟 |
| 18:00-19:00 | 晚餐 | 1h | 不要求健康，但要求至少吃 |
| 19:00-19:30 | **强制户外步行** | 30min | ★不可跳过(但可缩短至15min) |
| 19:30-21:00 | 缓冲时间 | 1.5h | 可以用来洗澡/发呆/洗衣服/喝中药 |
| 21:00-22:00 | 关机准备 | 1h | 如果失败了→仍然关机,原谅自己 |
| 22:00-07:30 | 睡眠 | 9.5h | 至少7.5h,多了更好 |

**缓冲机制说明：** 以上任何一个时间块都可以被突发事件打乱。计划的设计不依赖于你"完美执行"，而是依赖你"无论如何都能回到框架中"。

## 五、逐周进度说明

### 第1周（第1-7天）：戒断适应期

**核心任务：** 忍受戒断反应。戒断网文、游戏、色情的强烈渴望会出现。这是正常的生理反应，不是你的意志力弱。

| 天 | 预期感受 | 应对策略 |
|----|---------|---------|
| 1 | 焦虑、不安、"好想看小说" | 出门走15分钟 |
| 2 | 烦躁、注意力更差(比之前更差) | 这是戒断反应的高峰 |
| 3 | 可能出现头痛、情绪低落 | 正常。给自己泡杯热水 |
| 4-5 | 渴望开始减弱(但仍存在) | 初步的小胜利 |
| 6-7 | 出现"无聊"状态——不知道做什么 | ★这是好现象。无聊是大脑开始重置的信号 |

**第1周的"成功"标准：**
- ☐ 没有连续2天熬夜超过23:00
- ☐ 出门步行次数≥4次
- ☐ 自慰≤2次(不要求立即为零)
- ☐ 没有因为情绪崩溃而全天请假

### 第2周（第8-14天）：模式初见

**核心任务：** 开始感受到"不完美但持续执行"的力量。

- 睡眠开始有节律（不需要闹钟了，自然醒）
- 晚上9点放下手机仍然困难，但痛苦减轻
- "无聊感"持续，但不再恐慌
- 自慰冲动频率下降（但仍存在）
- 体力略有回升（上下楼不会喘了）

**重要警告：** 第2周是放弃的高峰期。你可能会想"这有什么用？我没感觉有什么变化"。坚持住——变化在第30天前后才开始显著。

### 第3周（第15-21天）：初步回弹

- 注意力的碎片化开始减少（能连续看一段文字10分钟而不跳）
- 下午精神顿困的频率降低
- 自慰可能已经减到1次/周
- 对网文的渴望变成"偶尔想起"而不是"时刻渴望"
- 与同事的对话开始多了一些（精力多了，话也多了一点）

### 第4周（第22-30天）：基础建立

- 睡眠节律基本稳定
- 每天的步行已经形成习惯（不需要意志力驱动）
- 自慰频率稳定在≤1次/周
- 对外界的感知开始复苏（开始注意到窗外的树、天气变化）
- "脑子清醒了一点"——前额叶开始重新供能

## 六、紧急应对方案

### 如果出现以下情况，按指示处理：

| 情况 | 处理方式 | 为什么 |
|------|---------|--------|
| 今天完全不想动 | 允许。躺在床上发呆。但手机放远 | 硬撑反而会制造"失败感" |
| 又陷入自慰后的自责 | 不行罚自己。说"好，下一次我再试试别的" | 自责是大脑让你放弃的陷阱 |
| 熬夜到凌晨3点 | 明天闹钟照常响。因了白天允许补觉 | 不要用"今天搞砸了明天也算了"的逻辑 |
| 又被网文吸进去3小时 | 关掉，然后出门走10分钟 | 打破"既然看了就看完"的完美主义 |
| 情绪崩溃想哭 | 哭。允许自己哭。然后洗脸 | 这是长期压抑的释放，不是软弱 |
| 开始质疑"这有什么用" | "30天后我再判断" | 你现在的判断是戒断反应的一部分，不是真实评估 |
| 身体突然不舒服 | 暂停一切计划，去看三甲医院的医生 | 健康优先 |

### "连续失败"规则

如果连续3天没有执行任何核心动作（步行、关机、早睡），说明计划设计不合理或你的状态需要更保守的调整。此时：

1. **不要自责。** 归因于计划设计问题，不是你的问题。
2. **降低门槛：** 把"步行30分钟"降为"出门站10分钟"；把"早睡"降为"22:30放下手机"
3. **重新开始。** 不设起点日期，任何一天都可以是第1天。

## 七、每日日志模版（养成记录习惯）

建议在 Obsidian 创建每日日志文件，格式如下：

```markdown
# 第X天 | 日期

## 执行情况（打勾）
- [ ] 步行15-30分钟
- [ ] 22:00前放下手机
- [ ] 22:30前熄灯
- [ ] 自慰：无
- [ ] 网文/游戏：不超过30分钟

## 今日感受（一句话）
[写在这里]

## 明日最小目标（只选1个）
[写在这里]
```

**这样做的好处：** 记录本身就是一种反馈。当你看到连续3天的勾，大脑会开始释放多巴胺——这是你当前缺乏的"正常激励回路"的重建训练。

## 八、30天后的预期交接棒

30天后（第31天），你应该达到的状态：

- ☐ 睡眠节律基本稳定（23:00前入睡 ≥ 20天）
- ☐ 步行习惯初步形成（日均≥4000步）
- ☐ 自慰频率降至≤1次/周
- ☐ 网文/游戏时间降至≤1h/天
- ☐ 能够做15分钟不碰手机的事（发呆/看书/拉伸都可以）
- ☐ 甲亢药物规律服用+至少一次复诊

达到这些标准后，进入第二阶段：90日认知重建计划。

**没有达到？延长本阶段。** 不设时间压力。Body基础没有打好之前，进入下一阶段是浪费时间。
